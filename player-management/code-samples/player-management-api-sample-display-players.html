개요

이 샘플 및 관련 코드는 프로덕션 개발을 위한 가이드로 제공됩니다. 코드는 프로덕션 용도로 인증되지 않았으며 공식적으로 지원되지 않습니다.

이 샘플 앱의 아키텍처, 특히 인증 및 AJAX 구현과 관련된 일반적인 접근 방식에 대해 알아보려면 다음을 참조하세요. HTTPRequest를 사용하는 예 .

앱 아키텍처

이 앱의 상위 수준 응용 프로그램 논리는 다음과 같습니다.

  1. 계정 액세스에 필요한 사용자 데이터 수집
  2. 버튼 클릭 시 AJAX 호출을 수행하여 모든 플레이어 정보 검색
  3. AJAX 호출 성공 시 다음을 수행합니다.
    • HTML 페이지에 JSON 응답 데이터 삽입
    • 플레이어 정보를 정렬된 목록에 표시하는 HTML을 동적으로 빌드하고 HTML 페이지에 삽입

코드펜 사용하기

다음은 위의 CodePen을 효과적으로 사용하기 위한 몇 가지 팁입니다.

  • 클릭하여 플레이어의 실제 표시를 전환합니다. 결과단추.
  • 클릭 HTML/CSS/JS버튼을 눌러 코드 유형 중 하나를 표시합니다.
  • 딸깍 하는 소리 CodePen에서 편집이 CodePen을 자신의 계정으로 포크하려면 오른쪽 상단 모서리에 있습니다.
