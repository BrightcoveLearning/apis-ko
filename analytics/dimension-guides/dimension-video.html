URL 매개변수

Analytics API 보고서는 다음 URL 매개변수를 지원합니다.

URL 매개변수
매개변수 설명 필수의 가치 기본
account 보고하려는 계정 예 쉼표로 구분된 목록으로 된 하나 이상의 계정 ID 없음
dimensions 보고할 측정기준 예 쉼표로 구분된 목록으로 된 하나 이상의 차원(일부 조합은 유효하지 않음) 없음
where 보고서에 대한 필터를 지정하는 데 사용됩니다. 아니요 {dimension} == {value} - 세미콜론으로 구분된 목록으로 하나 이상 없음
limit 반품할 상품 수 아니요 양의 정수 10
offset 건너뛸 항목 수 아니요 양의 정수 0
sort 항목을 정렬할 필드 아니요 요청에 대한 유효한 필드 video_view
fields 반환할 필드 아니요 보고하는 차원에 따라 다릅니다. video_view

reconciled 포함된 경우 결과를 기록 또는 실시간 데이터로 제한합니다. 아니요 사실 | 거짓 진실

to 요청 날짜 범위의 끝 아니요 ISO 8601 날짜(YYYY-MM-DD), 에포크 시간(밀리초), 현재 문자열 또는 상대 데이터(+7d). 지난 32일 이내의 날짜만 참여 끝점에 대해 또는 reconciled=false인 경우 허용됩니다. 지금

사용 가능한 필드

차원에 대해 다음 필드를 반환할 수 있습니다.

  • ad_mode_begin - 플레이어가 광고 모드에 들어간 횟수
  • ad_mode_complete - 플레이어가 광고 모드를 완료한 횟수
  • bytes_delivered - 플레이어 및 관련 미디어에 대해 클라이언트 시스템에 전달된 총 바이트
  • engagement_score - 비디오에 대한 참여 점수
  • play_request - 동영상 재생이 요청된 총 횟수
  • play_rate - 동영상의 평균 재생률(동영상 조회수를 노출수로 나눈 값)
  • video - 비디오 ID
  • video_duration - 동영상 길이(보고된 경우)
  • video_engagement_1 - 영상의 1% 지점에서의 조회수
  • video_engagement_100 - 동영상 재생 시간 100% 시점의 총 조회수
  • video_engagement_25 - 동영상 재생 시간의 25% 지점에서 총 조회수
  • video_engagement_50 - 동영상 재생 시간의 50% 지점에서 총 조회수
  • video_engagement_75 - 영상 길이 75% 지점에서의 총 조회수<
  • video_impression - 비디오가 플레이어에 로드된 총 횟수
  • video_name - 시간 프레임에서 가장 최근에 노출된 시간의 비디오 이름
  • video_percent_viewed - 모든 비디오 조회수에 대해 조회된 비디오 비율의 합계입니다. 예를 들어 해당 기간에 두 개의 보기가 있고 한 보기에서 본 비율이 78%이고 다른 보기에서 59%인 경우video_percent_viewed반환 될 것입니다 137.
  • video_seconds_viewed - 각 동영상 조회에 대한 총 조회 시간(초)
  • video_view - 동영상 재생이 시작된 횟수
  • video.reference_id - 비디오의 참조 ID
  • video.name - 비디오의 현재 이름
  • video.description - 비디오의 현재 설명
  • video.complete - 영상 처리 완료 여부
  • video.created_at - 동영상이 생성된 날짜/시간
  • video.duration - 비디오의 현재 길이
  • video.economics - 동영상의 광고 활성화 여부
  • video.long_description - 영상의 긴 설명
  • video.state - 비디오가 현재 있는지 여부ACTIVE또는INACTIVE
  • video.tags - 비디오의 태그
  • video.updated_at - 동영상이 마지막으로 수정된 날짜/시간
  • video.custom_fields.{field_name} - 이름이 지정된 비디오의 사용자 정의 필드 값{field_name}
  • video_download_request - 오프라인 재생이 활성화된 비디오에 대한 다운로드 요청 수
  • video_download_complete - 오프라인 재생이 가능한 동영상이 완전히 다운로드된 횟수
  • video_download_cancellation - 오프라인 재생이 가능한 동영상 다운로드가 취소된 횟수(다운로드가 완료되기 전에 페이지가 닫힌 횟수 포함)
  • video_download_error - 오류로 인해 오프라인 재생이 가능한 동영상 다운로드에 실패한 횟수

필터 값

필터로 사용되는 차원을 사용하면 결과를 좁힐 수 있습니다.

필터 값: 쉼표로 구분된 목록 또는 video.q==로 된 동영상 ID {video field} : {value}
