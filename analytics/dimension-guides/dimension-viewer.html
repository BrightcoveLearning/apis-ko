일부 계정에서는 뷰어 수준 분석을 사용할 수 없습니다. 계정에 대해 이 기능을 켜는 데 관심이 있으면 계정 관리자에게 문의하세요.

URL 매개변수

Analytics API 보고서는 다음 URL 매개변수를 지원합니다.

URL 매개변수
매개변수 설명 필수의 가치 기본
account 보고하려는 계정 예 쉼표로 구분된 목록으로 된 하나 이상의 계정 ID 없음
dimensions 보고할 측정기준 예 쉼표로 구분된 목록으로 된 하나 이상의 차원(일부 조합은 유효하지 않음) 없음
where 보고서에 대한 필터를 지정하는 데 사용됩니다. 아니요 {dimension} == {value} - 세미콜론으로 구분된 목록으로 하나 이상 없음
limit 반품할 상품 수 아니요 양의 정수 10
offset 건너뛸 항목 수 아니요 양의 정수 0
sort 항목을 정렬할 필드 아니요 요청에 대한 유효한 필드 뷰어_보기
fields 반환할 필드 아니요 보고하는 차원에 따라 다름 - 참조필드사용 가능한 필드는 아래를 참조하십시오. 뷰어_보기
format 결과를 반환할 형식 아니요 json | CSV | xlxs json
from 요청 날짜 범위의 시작 아니요 ISO 8601 날짜(YYYY-MM-DD), 에포크 시간(밀리초), 문자열 alltime 또는 상대 날짜(-1m). 지난 32일 이내의 날짜만 참여 끝점에 대해 또는 reconciled=false인 경우 허용됩니다. 지금으로부터 30일 전
to 요청 날짜 범위의 끝 아니요 ISO 8601 날짜(YYYY-MM-DD), 에포크 시간(밀리초), 현재 문자열 또는 상대 데이터(+7d). 지난 32일 이내의 날짜만 참여 끝점에 대해 또는 reconciled=false인 경우 허용됩니다. 지금

사용 가능한 필드

차원에 대해 다음 필드를 반환할 수 있습니다.

  • viewer - 이벤트 뷰어 (SSO ID 또는 장치 ID 또는 사용자 에이전트 및 IP 기반 고유 문자열)
  • video_view - 뷰어 재생이 시작된 횟수
  • time - 비디오 보기의 에포크 시간(밀리초)
  • engagement_score - 참여 점수

필터 값

필터로 사용되는 차원을 사용하면 결과를 좁힐 수 있습니다.

필터 값: 뷰어 ID를 쉼표로 구분된 목록 또는 뷰어.q== {viewer field} : {value}
