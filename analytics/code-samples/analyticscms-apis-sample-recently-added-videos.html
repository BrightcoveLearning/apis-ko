제한 사항

이 샘플 및 관련 코드는 프로덕션 개발을 위한 가이드로 제공됩니다. 코드는 프로덕션 용도로 인증되지 않았으며 공식적으로 지원되지 않습니다.

사용된 API

이 샘플은 두 가지 Brightcove API를 사용합니다.

  • CMS API (최근에 추가된 비디오를 찾기 위해)
  • Analytics API(분석 가져오기)

자격 증명 얻기

얻으려면client_id그리고client_secret , OAuth UI로 이동하여 이 앱을 등록해야 합니다.

  • API 인증 자격 증명 관리

필요한 권한은 다음과 같습니다.

CURL 또는 Postman을 통해 자격 증명을 얻을 수도 있습니다. 다음을 참조하세요.

  • CURL을 사용하여 클라이언트 자격 증명 가져오기
  • Postman을 사용하여 클라이언트 자격 증명 가져오기

API에서 직접 자격 증명을 받는 경우 필요한 권한은 다음과 같습니다.

            [
              "video-cloud/analytics/read",
