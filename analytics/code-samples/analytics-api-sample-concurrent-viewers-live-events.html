소개

이 샘플은라이브 분석 시계열 가져오기끝점Analytics API다른 시간 간격을 기반으로 한 라이브 이벤트의 동시 시청자에 대한 보고서를 생성합니다. 편의를 위해 API에서 반환된 Epoch 타임스탬프는 사람이 읽을 수 있는 타임스탬프로 변환됩니다.

제한 사항

이 샘플 및 관련 코드는 프로덕션 개발을 위한 가이드로 제공됩니다. 코드는 프로덕션 용도로 인증되지 않았으며 공식적으로 지원되지 않습니다.

자격 증명 얻기

얻으려면client_id그리고client_secret , OAuth UI로 이동하여 이 앱을 등록해야 합니다.
