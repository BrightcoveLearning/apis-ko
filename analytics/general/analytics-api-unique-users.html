데이터 가용성

고유 추정치는 일괄 처리 시스템에서 발생하므로 이 daily_unique_viewers 값을 실시간으로 사용할 수 없습니다. Daily_unique_viewers의 값은 측정되는 날짜가 지난 후 3-5일 동안 사용할 수 없습니다. API를 호출하고 NULL 응답을 받으면 데이터를 아직 사용할 수 없음을 의미합니다.

JSON은 NULL을 반환하고 CSV는 ""를 반환합니다. CSV에는 null 값을 표시하는 방법이 없습니다.

사용자 식별

Video Cloud Analytics는 다음으로 구성된 문자열을 사용합니다. Source IP address + the User-Agent String사용자의 고유 식별자로 사용됩니다.

총 고유 사용자

모든 사용자의 고유 식별자를 저장하는 것은 불가능하기 때문에 알고리즘을 사용하여 계정의 총 사용자를 추정합니다.

즉, 정확한 시청자 수를 알고 이를 당사의 수와 비교하면 약간의 차이가 있음을 알 수 있습니다. 이 오류율은 순 시청자 수가 증가할수록 증가합니다.
