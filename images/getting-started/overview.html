소개

Brightcove Image API는 간단한 GET 요청을 사용하여 이미지 크기를 조정, 자르기 및/또는 회전하는 빠르고 쉬운 방법을 제공합니다.

요청 URL

이미지 API에 대한 요청 URL은 다음과 같습니다.

https://images.brightcovecdn.com/image/v1/account_id/default/url/{image_url}

필수 경로 매개변수는 다음과 같습니다.

  • account_id : 귀하의 Video Cloud 계정 ID
  • image_url : 이미지를 찾을 수 있는 공개 URL
노트:
  • 다음 이미지 형식이 지원됩니다. PNG, JPG, GIF
  • NS image_url제대로 인코딩해야 합니다. JavaScript를 사용하는 경우 다음을 사용하십시오. encodeURIComponent()방법. 당신은 또한 사용할 수 있습니다문자열 인코더 Brightcove 학습 서비스 제공:

URL 매개변수

이미지의 변형된 버전을 얻으려면 요청에서 아래에 설명된 하나 이상의 URL 매개변수를 사용하십시오. 3개의 매개변수를 조합하여 사용할 수 있습니다.

이미지 API용 URL 매개변수
매개변수 설명
resize 이것을 "widthxheight"와 동일하게 설정하여 이미지 크기를 조정합니다. 예: 크기 조정=160x90 )
crop 이것을 "widthxheight"와 동일하게 설정하여 너비와 높이를 자릅니다. 예: 자르기=160x90 - 원점은 이미지의 중심입니다.
rotate 지정된 각도만큼 이미지를 시계 방향으로 회전 - 예: 회전=90

모범 사례

변환된 이미지가 웹 페이지나 온라인 앱에 표시될 경우 API를 사용하여 이미지를 검색하고 서버에 저장합니다. 하다 ~ 아니다 API의 속도 제한으로 인해 트래픽이 많은 페이지 또는 앱에서 일부 요청이 실패하고, 로드될 때마다 API 요청이 이루어지도록 하여 페이지/앱의 성능을 저하시키므로 런타임 시 검색합니다. 사용자.
