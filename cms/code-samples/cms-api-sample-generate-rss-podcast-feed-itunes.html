소개

이 앱은 팟캐스트를 만듭니다 iTunes용 RSS 피드 Video Cloud 계정에 있는 하나 이상의 오디오 전용 타이틀에서.

이 샘플 및 관련 코드는 프로덕션 개발을 위한 가이드로 제공됩니다. 코드는 프로덕션 용도로 인증되지 않았으며 공식적으로 지원되지 않습니다.

참고: 모든 필수 정보를 입력했지만 피드가 생성되지 않거나 동영상이 포함되지 않은 경우문제 해결아래 섹션.

이 앱을 사용하기 위해 필요한 것:

  • 검색에 반환된 모든 비디오는 피드에 포함됩니다. 따라서 태그 또는 사용자 정의 필드를 사용하여 포함하려는 비디오를 정확히 식별해야 합니다.
  • 피드의 URL ~해야하다 MP4 변환을 가리킵니다. 선택한 비디오에 MP4 변환이 없는 경우 앱은 ~ 아니다피드에 포함합니다.
  • 당신이 만족하는지 확인하십시오애플의 요구사항이미지 및 기타 사항에 대해 이 앱은 팟캐스트 아트워크에 포스터 이미지를 사용합니다. 사용자 지정을 만들어야 할 수도 있습니다. 프로필 수집포스터 크기를 이미지 요구 사항에 맞게 만듭니다.
  • 또한 참조팟캐스트 모범 사례 (이 앱은 입력이 허용하는 한 이를 따릅니다).
  • 귀하의 팟캐스트가 오디오 전용 콘텐츠로 비디오 클라우드에 추가됩니다. - 참조오디오 콘텐츠이 작업을 수행하는 방법에 대한 자세한 내용은
  • 팟캐스트의 제목입니다. 제목이 지정되지 않은 경우name비디오(또는 에피소드 그룹의 첫 번째 비디오)가 사용됩니다.
  • 팟캐스트에 대한 설명입니다.
  • 사이트 URL(예: https://yoursiteurl.com)
  • 이 피드가 호스팅될 URL(예: https://yoursiteurl.com/feeds/podcast.rss)
  • 작성자(개인 또는 조직 이름).
  • 팟캐스트의 연락처 이메일 주소
  • 방송 요약(생략하면description비디오 클라우드에서).

아래 앱에는 제공해야 하는 모든 정보에 대한 입력 필드가 있습니다. 앱이 데이터 및 비디오 메타데이터를 팟캐스트 피드 필드에 매핑하는 방법을 보려면필드 매핑섹션을 참조하십시오.

URL 토큰화

위의 요구 사항 외에도 계정의 URL 토큰화를 수정해야 합니다. 비디오 클라우드는ttl콘텐츠 도난을 방지하기 위해 변환용 URL에 대한 토큰. 기본적으로 이러한 토큰은 수명이 매우 짧습니다. 팟캐스트에는 너무 짧습니다. 당신은해야 할 것입니다 Brightcove 지원에 문의토큰 수명 값을 더 높게 설정하도록 합니다. 허용 범위는 1시간에서 1년입니다. 팟캐스트를 사용할 수 있기를 원하는 시간에 적절한 값으로 설정하도록 요청하십시오. 그러나 이것은 계정 설정임을 기억하십시오. 다른 비디오의 보안을 위태롭게 할 수 있도록 값을 너무 높게 설정하지 마십시오.

앱 작동 방식에 대한 참고 사항

  • 피드에 필요한 정보를 입력한 후 계정에서 동영상을 가져옵니다(최대 20개).
  • 팟캐스트 비디오에 대한 이름을 지정하거나 태그를 지정하거나 사용자 정의 필드 값이 있으면 쉽게 찾을 수 있습니다.
  • 그런 다음 피드에 대해 하나 이상의 비디오를 선택합니다.
  • 여러 비디오를 선택하는 경우 앱은 해당 비디오가 팟캐스트 에피소드이고 비디오의 활성화 날짜가 에피소드 순서를 반영한다고 가정합니다.

문제 해결

이 앱에 문제가 있는 경우 다음 사항을 확인하세요.

  • 생성된 피드에 비디오가 없는 경우 포함하려는 비디오에 MP4 변환이 있는지 확인하십시오. CMS API를 사용하여 이를 확인할 수 있지만 가장 쉬운 방법은 Studio Media 모듈에서 비디오 파일 목록을 보고 최소한 하나의 변환을 다운로드할 수 있는지 확인하는 것입니다.

    다운로드할 수 있는 변환이 없으면 MP4(프로그레시브) 변환이 포함된 수집 프로필을 사용하여 비디오를 교체하거나 다시 코드 변환해야 합니다.

  • 비디오에 MP4 변환이 있지만 피드에 여전히 비디오가 없는 경우 사용 중인 검색 문자열이 올바른 비디오를 반환하는지 확인하십시오. 이를 수행하는 한 가지 방법은 아래 요청 URL을 복사하고{account_id}그리고{search_string}귀하의 계정 ID와 검색 문자열을 사용하여 API 테스터예상한 동영상을 반환하는지 확인하려면 다음 단계를 따르세요.

            https://cms.api.brightcove.com/v1/accounts/{account_id}/videos?q={search_string}
