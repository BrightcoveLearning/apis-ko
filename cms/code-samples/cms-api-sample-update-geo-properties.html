이 앱의 제한 사항

이 샘플 및 관련 코드는 프로덕션 개발을 위한 가이드로 제공됩니다. 코드는 프로덕션 용도로 인증되지 않았으며 공식적으로 지원되지 않습니다.
  • 앱은 동일한 지역 설정으로 계정의 모든 비디오를 업데이트합니다. 폴더 또는 일부 검색 기준으로 요청하여 비디오의 일부만 선택하는 옵션을 추가하는 것은 매우 간단합니다.
  • 예외 처리가 없습니다. 여기에 사용된 것과 같은 작은 계정의 경우 크게 중요하지 않습니다. API 요청 시간이 초과되면 앱이 그 시점에서 실패하고 간단히 다시 실행할 수 있습니다. 큰 계정의 경우 예외 처리를 추가하고 싶을 것입니다. 최소한offset처리된 동영상 수를 추적하는 변수로 해당 시점에서 앱을 다시 시작할 수 있습니다.
  • 대규모 계정의 경우 처리를 완료하는 데 시간이 걸립니다. 앱이 처리를 시작한 후에 추가된 모든 비디오는 누락됩니다. 그러나 앱은 처리한 동영상 수를 알려 주므로 나중에 누락된 동영상이 있는지 확인하기 위해 다른 수의 동영상을 얻을 수 있습니다. 결과는 생성 날짜 오름차순으로 정렬되므로 놓친 동영상이 계정에 마지막으로 추가된 동영상이 됩니다.

자격 증명 가져오기

사용하려면 CMS API적절한 자격 증명이 필요합니다.

대부분의 경우 자격 증명을 얻는 가장 쉬운 방법은 Studio Admin API 인증 섹션을 사용하는 것입니다(계정에 대한 관리자 권한 필요). 보다 API 인증 자격 증명 관리자세한 내용은. 대부분의 경우 모든 권한을 얻기를 원할 것입니다. CMS API작업:

필요한 권한을 Studio에서 사용할 수 없거나 OAuth API에서 직접 권한을 가져오려면 원하는 권한을 사용하십시오. 클라이언트 자격 증명 가져오기아래 나열된 문서. 어떤 옵션을 선택하든 올바른 작업 권한을 요청해야 합니다. 다음을 cURL 또는 Postman과 함께 사용하여 CMS API에 대한 모든 권한을 얻을 수 있습니다.
