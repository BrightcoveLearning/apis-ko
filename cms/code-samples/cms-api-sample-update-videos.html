소개

NS CMS API단일 요청으로 여러 동영상 업데이트를 지원하지 않습니다. 그러나 이 작업을 수행하기 위해 프로그래밍 방식으로 비디오 데이터 배열을 반복할 수 있으며 이 샘플은 샘플 Brightcove 계정을 사용하여 이를 보여줍니다. 액세스 토큰을 가져오고 API를 호출하기 위해 Brightcove Learning Services에서 구축한 프록시를 사용합니다.

목표는 논리를 보여주는 것이므로 비디오에 대한 메타데이터의 한 비트인 설명만 업데이트합니다.

이 샘플 및 관련 코드는 프로덕션 개발을 위한 가이드로 제공됩니다. 코드는 프로덕션 용도로 인증되지 않았으며 공식적으로 지원되지 않습니다.
