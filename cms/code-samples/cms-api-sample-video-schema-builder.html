사용 및 제한 사항

이 샘플 및 관련 코드는 프로덕션 개발을 위한 가이드로 제공됩니다. 코드는 프로덕션 용도로 인증되지 않았으며 공식적으로 지원되지 않습니다.

참고: 이 샘플 앱에서 제공하는 기능은 이제비디오 SEO 스키마 생성기 플러그인 . 가능하면 대신 사용하는 것이 좋습니다.

여기에서 코드를 생성하고 단일 코드를 사용하는 경우 게시 코드로 사용할 수 있습니다. 브라이트코브 플레이어계정 및 표준(iframe) 포함 다른 플레이어 또는 고급(인페이지) 포함을 사용하는 경우 생성된 코드의 스키마 부분을 사용하여 페이지의 플레이어 포함 코드 위에 삽입할 수 있습니다.

또한 이 앱은 단일 비디오 플레이어용 코드를 생성하며 스키마 코드를 수동으로 포함하는 경우 URL이 변경될 수 있으므로 주기적으로 업데이트해야 합니다.

자격 증명 가져오기

사용하려면 CMS API적절한 자격 증명이 필요합니다.

대부분의 경우 자격 증명을 얻는 가장 쉬운 방법은 Studio Admin API 인증 섹션을 사용하는 것입니다(계정에 대한 관리자 권한 필요). 보다 API 인증 자격 증명 관리자세한 내용은. 대부분의 경우 모든 권한을 얻기를 원할 것입니다. CMS API작업:

필요한 권한을 Studio에서 사용할 수 없거나 OAuth API에서 직접 권한을 가져오려면 원하는 권한을 사용하십시오. 클라이언트 자격 증명 가져오기아래 나열된 문서. 어떤 옵션을 선택하든 올바른 작업 권한을 요청해야 합니다. cURL 또는 Postman과 함께 다음을 사용하여 모든 권한을 얻을 수 있습니다. CMS API :

        "operations": [
          "video-cloud/video/all",
          "video-cloud/playlist/all",
          "video-cloud/sharing-relationships/all",
          "video-cloud/notifications/all"
        ]

  • 인증: Postman을 사용하여 클라이언트 자격 증명 가져오기
