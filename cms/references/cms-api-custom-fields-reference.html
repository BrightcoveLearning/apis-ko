사용자 정의 필드

들 유형 설명
max_custom_fields 숫자 게시자가 계정에서 만들 수 있는 사용자 정의 필드 수
custom_fields 정렬 계정의 사용자 정의 필드 개체 목록
standard_fields 정렬 필수이거나 설명이 있는 필드 목록입니다. 둘 다 없는 표준 필드는 이 배열에 나타나지 않습니다.

사용자 정의 필드 맵

각 사용자 정의 필드에 대해:

들 유형 설명
id 끈 필드의 내부 이름
description 끈  
display_name 끈  
type 끈 열거형 중 하나' 또는 '문자열' (이것은 '목록' 또는 '텍스트' UI에서)
enum_values 정렬 사용자 정의 필드에 유효한 문자열 값의 배열. type = 'enum'일 때만 존재
required 부울 비디오가 활성화되기 전에 이 필드에 값이 있어야 하는지 여부
