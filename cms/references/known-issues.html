원본 파일 이름이 손상됨

어떤 경우에는original_filename비디오 필드가 손상되고 실제 파일 이름이 포함되지 않습니다.

태그는 쉼표를 포함할 수 없습니다.

쉼표(예: "SomeCo, Inc") "불법 태그" 오류와 함께 요청이 실패합니다.

예약된 동영상

Playback API 및 Player Catalog는 최대 10-15분 동안 비디오를 캐시하기 때문에 다음 몇 분(최대 20분) 동안 사용할 수 있도록 예약된 비디오를 요청하는 플레이어는 캐시를 새로 고칠 때까지 재생 가능한 비디오를 얻지 못할 수 있습니다. .

디지털 마스터 있음

이 필드를 사용하여 비디오를 재변환할 수 있는지 여부를 테스트하는 경우 원본 비디오에 마스터가 있는 경우에도 마찬가지이므로 공유 비디오에 대해서는 신뢰할 수 없지만 여전히 제휴 계정으로 재변환할 수는 없습니다.

해결 방법

동영상이 계정에 공유된 경우 두 가지 모두에 대해 테스트해야 합니다. has_digital_master === true그리고sharing.by_external_account === false . 샘플 코드(JavaScript):

var video={some_video_object}
if (video.sharing !== null) {
  if (video.sharing.by_external_account === false) {
    if (video.has_digital_master) {
    // video can be retranscoded
    }
  }
}
