비디오 필드

비디오를 만들거나 업데이트할 때 설정할 수 있는 필드를 보려면 다음 링크를 참조하십시오.

  • 동영상 제작 시 설정할 수 있는 필드
  • 동영상 업데이트 시 설정할 수 있는 필드
비디오 필드
들 유형 설명
account_id 끈 비디오 클라우드 계정 ID
ad_keys 끈 동영상에 할당된 광고 키/값 쌍을 나타내는 문자열입니다. 키/값 쌍은 키=값 형식으로 지정되며 앰퍼샌드로 구분됩니다. 예를 들어: "adKeys": "category=sports&live=true"
clip_source_video_id 끈 이 비디오를 제작하기 위해 잘린 소스 비디오의 ID 또는null이 비디오가 다른 비디오의 클립이 아닌 경우.
complete 부울 처리 완료 여부 — 참고: 새 비디오를 만들 때 complete 속성은 자동으로 다음으로 설정됩니다. false . 비디오에 대한 하나의 변환이 존재하는 즉시 complete 속성이 다음으로 설정됩니다. true
created_at 숫자, 에포크로부터의 밀리초 비디오 클라우드 시스템에서 비디오가 생성되었을 때
만든 사람 물체 비디오를 만든 사람에 대한 정보(가능한 경우)
큐 포인트 지도 배열 큐 포인트 맵의 배열
custom_fields 필드-값 쌍의 맵(문자열) 비디오에 대해 설정된 사용자 정의 필드 값 - ~ 아니다이 비디오에 대한 값이 이 지도에 포함되어 있지 않습니다. 사용자 정의 필드 값의 최대 길이는 1바이트 1024자입니다.
dash_manifest_id 끈 DASH 매니페스트의 시스템 ID
delivery_type 끈 동영상 전송 유형:
  • dynamic_origin - 동적 전달
  • live_origin - Brightcove Live의 라이브 스트림
  • remote - 자산
  • static_origin - 레거시 수집 시스템
  • unknown - 결정할 수 없음(비디오에 재생 가능한 변환이 없음을 의미할 수 있음)
description 끈; 이전 shortDescription을 대신합니다. 비디오에 대한 간단한 설명(최대 길이: 1바이트 문자 248자)
digital_master_id 끈 디지털 마스터의 시스템 ID
drm_disabled 부울 DRM 사용 계정의 경우 이것을 true로 설정하면 비디오가 DRM으로 패키징되지 않습니다. 계정이 DRM을 사용할 수 없는 경우 이 필드가 표시되지 않습니다.
duration 숫자, 밀리초 비디오 재생 시간(밀리초)
economics 문자열, 유효한 열거형 값 중 하나여야 합니다. "AD_SUPPORTED"(기본값) 또는 "FREE"
folder_id 끈 동영상이 속한 폴더의 ID
forensic_watermarking 끈 ACTIVE비디오에 포렌식 워터마킹이 있음을 나타냅니다. UNAVAILBLE포렌식 워터마킹이 추가되지 않았음을 의미합니다. null계정이 포렌식 워터마킹에 대해 활성화되지 않았음을 나타냅니다.
지리적 [1-3] 속성-값 쌍의 지도 동영상의 지역 제한 속성
has_digital_master 부울 비디오에 재변환에 사용할 수 있는 보관된 마스터가 있는지 여부입니다. 이 필드를 사용하여 비디오를 다시 트랜스코딩할 수 있는지 여부를 테스트하는 경우 공유 비디오에 대해 신뢰할 수 없습니다. 동영상이 계정에 공유된 경우 두 가지 모두에 대해 테스트해야 합니다. has_digital_master = true그리고sharing.by_external_acct = false
id 끈 동영상 아이디
이미지 속성-값 쌍의 지도 포스터 및 썸네일 이미지에 대한 정보 맵
링크 속성-값 쌍의 지도 관련 링크 속성의 지도
long_description 끈 긴 설명(최대 5000자)
name 끈 동영상 이름(최대 길이: 1바이트 248자)
offline_enabled 부울 비디오가 오프라인 재생을 위해 활성화되었는지 여부
original_filename 끈 수집된 비디오 파일의 이름 - 원격 자산 비디오의 경우 null입니다.
projection 끈 360° 비디오에 대한 매핑 투영(예: "정방형")
published_at 끈 비디오가 게시된 시간을 나타내는 ISO-8601 날짜-시간 문자열 - 이 필드는 쓰기 가능합니다. 기본적으로 비디오가 처음 활성화된 날짜로 설정됩니다.
reference_id 끈 사용자 지정 ID 비디오를 고유하게 식별 , 150자로 제한됩니다. referenceId는 다른 시스템에서 이 비디오를 식별하기 위한 외래 키로 사용할 수 있습니다. 참조 ID는 공백, 쉼표 또는 특수 문자를 포함할 수 없습니다.
일정 [1-1] 속성-값 쌍의 지도 비디오 가용성에 대한 시작 및 종료 날짜-시간 지도
sharing 속성-값 쌍의 지도 비디오의 공유 속성 맵
smooth_client_manifest_id 끈 Smooth 매니페스트의 시스템 ID
smooth_server_manifest_id 끈 Smooth 매니페스트 서버의 시스템 ID
state [1-4] 끈 활성, 비활성, 보류 중, 삭제됨
tags [1-5] 태그 배열(문자열) 비디오에 할당된 태그 배열
text_tracks HTML5 스타일 텍스트 트랙 배열 비디오에 할당된 텍스트 트랙(WebVTT 파일) 배열
updated_at 날짜-시간 문자열 동영상이 마지막으로 수정된 날짜-시간 [1-2]
업데이트됨_by 물체 비디오를 마지막으로 업데이트한 사람에 대한 정보(가능한 경우)
변종 의 배열variant메타데이터 개체 다양한 언어에 대한 메타데이터 변형

노트

  • [1-1] 예약된 동영상은 ~ 아니다예정된 이용 가능 시간 외 비활성
  • [1-2] 사용자 또는 Video Cloud 시스템이 수정했을 수 있습니다.
  • [1-3] 지리적 제한은 Video Cloud에 대한 유료 애드온입니다. 지역 필터링 기능을 구매하려면 계정 관리자에게 문의하세요.
  • [1-4] state로 설정할 수 있습니다ACTIVE또는INACTIVE , 하지만 ~ 아니다에게PENDING또는DELETED .
  • [1-5] 태그에 대한 다음 제한 사항에 유의하십시오.
    • 태그는 최대 128자까지 가능합니다. 태그가 128자를 초과하는 경우 변경 사항을 저장하려고 할 때 오류가 표시됩니다.
    • 태그에는 쉼표를 사용할 수 없습니다. 예를 들어roof, attic + front yard지원되지 않습니다
    • 동영상은 총 1200개 이하의 태그를 포함할 수 있습니다.
    • 대소문자(소문자, 대문자)만 다른 태그는 동일한 태그로 처리되고 대소문자는 자동으로 원래 태그의 태그와 일치합니다. 대소문자 구분 문제는 예를 들어 JavaScript 메소드를 사용하여 수정하는 등 Video Cloud 외부에서 처리해야 합니다. 문자열
