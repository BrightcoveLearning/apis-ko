    CMS API 비디오 필드 참조

    이것은 CMS API에서 반환한 비디오 필드에 대한 참조입니다.

    비디오 필드

    비디오를 만들거나 업데이트할 때 설정할 수 있는 필드를 보려면 다음 링크를 참조하십시오.

    • 비디오를 만들 때 설정할 수 있는 필드
    • 동영상 업데이트 시 설정할 수 있는 필드
    동영상 필드
    필드 유형 설명
    account_id 문자열 비디오 클라우드 계정 ID
    ad_keys 문자열 동영상에 할당된 광고 키/값 쌍을 나타내는 문자열입니다. 키/값 쌍은 키 = 값으로 형식이 지정되며 앰퍼샌드로 구분됩니다. 예를 들면 다음과 같습니다. "adKeys": "category=sports&live=true"
