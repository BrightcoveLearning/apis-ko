소개

NS CMS API Video Cloud 카탈로그에서 디지털 마스터(비디오의 소스 파일)를 삭제하는 기능이 있습니다. 이렇게 하면 관리되는 콘텐츠의 크기를 줄이기 위해 일회성 정리 작업을 수행할 수 있습니다.

마스터가 아카이브되는 방법과 마스터가 아카이브되지 않도록 방지하는 방법을 이해하려면 - 이것이 원하는 경우 - 참조아카이빙 마스터 .

요구 사항

  • 디지털 마스터 삭제는 다음을 사용하여 수행할 수 있습니다. CMS API오직 - 이 옵션은 Studio에서 사용할 수 없습니다.

마스터 삭제의 의미

계속하기 전에 디지털 마스터 삭제의 다음 의미를 이해해야 합니다.

  • 마스터가 삭제된 후에는 더 이상 비디오를 다시 코드 변환할 수 없습니다. 새 변환이 필요한 경우 다음을 통해 비디오를 교체해야 합니다. 사진관아니면 그Dynamic Ingest API .
  • 일괄 프로비저닝(더 이상 사용되지 않음)을 통한 비디오 관리는 더 이상 작동하지 않습니다. 사용 CMS그리고동적 수집대신 API.
  • 디지털 마스터를 삭제하는 것은영구적 인작업. 우리할 수 없다삭제된 디지털 마스터를 복원합니다.

단계: 디지털 마스터 삭제

디지털 마스터를 삭제하려면 다음 단계를 따르십시오.

  1. 확실히하다당신은 자신의 마스터 사본을 가지고 있습니다당신이 그것을 필요로하는 경우에.
  2. 비디오를 재변환해야 하는 경우 마스터를 삭제하기 전에 재변환하십시오. 마스터 삭제 후더 이상 비디오를 다시 코드 변환할 수 없습니다.
  3. 작업에 필요한 클라이언트 자격 증명을 얻습니다. 클라이언트 자격 증명에는 다음 작업에 대한 권한이 포함되어야 합니다. 
          [
          "video-cloud/asset/delete",
          "video-cloud/video/read"
      ]

    이러한 권한은 ~ 아니다 Studio UI에서 사용할 수 있으므로 OAuth API그들을 얻기 위해. 다음은 강조 표시된 항목을 자신의 정보로 교체하는 경우 작업을 수행하는 cURL 명령입니다.

          curl --request POST \
        --url https://oauth.brightcove.com/v4/client_credentials \
        --header 'authorization: BC_TOKEN YOUR_BC_TOKEN' \
        --header 'content-type: application/json' \
        --data '{
        "type": "credential",
        "maximum_scope": [
          {
            "identity": {
              "type": "video-cloud-account",
              "account-id": YOUR_ACCOUNT_ID
            },
            "operations": [
          "video-cloud/asset/delete",
          "video-cloud/video/read"
      ]
          }
        ],
        "name": "Delete-Masters"
      }'

    BC_TOKEN을 얻으려면 Studio 페이지를 열고(계정의 관리자여야 함) 브라우저에서 JavaScript 콘솔을 열고 아래 Javascript 코드를 붙여넣고 Enter 키를 누릅니다.

          var cookiesArray = document.cookie.split(";"), cookiesObj = {}, i, tmpArray = [];
      for (i = 0; i < cookiesArray.length; i++) {
          tmpArray = cookiesArray[i].split("=");
          if (tmpArray[0].indexOf('BC_TOKEN') > -1) {
              cookiesObj.BC_TOKEN = tmpArray[1];
          }
      }
      window.prompt("BC_TOKEN:", cookiesObj.BC_TOKEN);
  4. 클라이언트 자격 증명을 사용하여액세스 토큰 생성
  5. 이 끝점을 사용하여 디지털 마스터를 삭제합니다.
    마스터 삭제
    기본 URL 
          https://cms.api.brightcove.com/v1
    방법 DELETE
    끝점 
          /accounts/accountID/videos/videoID/digital_master
    헤더 
          Authorization: Bearer Authorization: Bearer {access_token}
    응답 204 NO_CONTENT(204는 삭제 작업에 대한 표준 응답으로 작업이 성공했거나 자산이 이미 삭제되었음을 나타냅니다.)
