개요

비디오를 360° 비디오로 식별하려면 투영 속성을 설정해야 합니다. 이 속성을 설정할 수 있습니다. Studio에서 미디어 모듈 사용또는 이 항목에 표시된 대로 프로그래밍 방식으로 수행할 수 있습니다.

이 예에서는 Postman 앱을 사용하여 CMS API로 비디오 속성을 업데이트합니다. 시작하는 방법 알아보기 HTTP 요청에 Postman 사용문서.

이 항목에서는 다음 작업을 수행합니다.

  • 클라이언트 자격 증명 가져오기
  • 비디오 정보 검색
  • 360° 비디오 속성 설정

클라이언트 자격 증명 가져오기

Brightcove API로 작업하려면 사용하려는 계정 및 API에 대한 클라이언트 자격 증명이 필요합니다.

  1. 의 지침에 따라 Studio에서 클라이언트 자격 증명을 가져옵니다. API 인증 자격 증명 관리 . 아래 단계에서는 Postman을 사용하여 Player Management API를 요청할 것이므로 자격 증명에는 최소한 다음 권한이 있어야 합니다.

    • CMS : 비디오 읽기/쓰기

    더 넓은 범위의 API 요청에 사용할 수 있는 자격 증명을 얻기 위해 원하는 만큼 추가 권한을 추가할 수 있습니다. 또한 원하는 경우 여러 계정에서 사용할 수 있는 자격 증명을 얻을 수 있습니다.

    클라이언트 자격 증명을 생성하려면 Studio에 대한 관리자 권한이 있거나 조직에서 관리자 권한이 있는 사람이 자격 증명을 생성하도록 해야 합니다.

비디오 정보 검색

비디오를 Video Cloud Studio에 업로드하면 CMS API에서 비디오 정보를 검색할 준비가 된 것입니다. 비디오 업로드에 대한 자세한 내용은업로드 모듈을 사용하여 비디오 업로드문서.

  1. 우편 배달부 앱을 엽니다.
  2. Video Cloud 계정에 대한 새 액세스 토큰을 받고 다음을 선택하십시오. 토큰 사용단추.
  3. 을 만들다가져 오기다음을 사용하여 CMS API에 요청 ID로 비디오 가져오기통사론. 다음 URL을 입력하면 됩니다. 요청 URL 입력필드, 계정 ID 대체{account_id}및 귀하의 비디오 ID {video_id} : 
        https://cms.api.brightcove.com/v1/accounts/{account_id}/videos/{video_id}
  4. 방법을 GET으로 두고 다음을 선택합니다. 보내다 :
  5. 응답 본문에서 다음을 확인해야 합니다. 투사재산.

360° 비디오 속성 설정

비디오가 360° 비디오임을 나타내려면투사속성을 다음으로 설정해야 합니다. 정방형 .

  1. Postman 앱으로 돌아갑니다. 새 액세스 토큰을 받아야 할 수도 있습니다.
  2. 선택하다반점이전 섹션에서와 동일한 URL 구문을 사용하여 HTTP 메소드로.
  3. 클릭헤더버튼을 누르고핵심 가치쌍콘텐츠 유형 - 애플리케이션/json . 단순히 입력 영역에 입력을 시작하면 표시된 값을 선택할 수 있는 옵션이 제공됩니다.
  4. 클릭신체 URL 바로 아래 섹션을 클릭한 다음날것의라디오 버튼:
  5. 원시 데이터의 경우 본문에 다음 JSON 코드를 입력합니다(JSON 다음의 스크린샷은 요청이 표시되는 방식을 보여줍니다). 
        {
        "projection" : "equirectangular"
    }
  6. 딸깍 하는 소리보내다
  7. 응답 본문에서 이제투사속성이 설정되었습니다.
