소개

더 이상 관련이 없거나 다른 이유로 동영상을 삭제해야 할 수도 있습니다. 이것은에서 할 수 있습니다사진관또는 아래에 설명된 대로 CMS API를 사용합니다.

프로세스는 간단하지만 다음과 같은 몇 가지 사항을 알고 있어야 합니다.

  • 동영상 삭제는 취소할 수 없습니다.
  • 캐싱으로 인해 삭제된 비디오는 Video Cloud 시스템이나 인터넷에서 즉시 사라지지 않습니다. 비디오를 즉시 사용할 수 없게 만드는 유일한 방법은 모든 플레이어에서 비디오를 제거하는 것입니다.
  • 여러 동영상을 삭제할 수 있지만 한 번에 삭제할 수 있는 수는 API 요청에 대한 URL의 최대 길이에 의해 제한됩니다. 일반적으로 한 번의 요청으로 최대 100개까지 삭제할 수 있습니다.

API 요청 세부정보

끝점

/v1/accounts/{account_id}/videos/{video1_id},{video2_id}...

HTTP 메소드

DELETE

입증

액세스 토큰, 모든 것과 동일 CMS API요구 .

요청 본문

없음.

응답

204 No Content

노트
