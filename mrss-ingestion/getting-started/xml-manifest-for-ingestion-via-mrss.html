페이지 내용

    MRSS를 통한 수집을 위한 XML 매니페스트

    이 항목에서는 비디오를 Video Cloud에 업로드하기 위해 MRSS 수집에서 사용하는 XML의 구조에 대해 설명합니다. 참고: 베타 수집 서비스입니다.

    서문

    비디오를 인제스트할 때 XML 매니페스트 파일을 사용하여 추가 메타데이터 (비디오 제목, 설명, 비행 시간, 레이블 및 사용자 정의 메타데이터 포함) 뿐만 아니라 축소판 및 폐쇄 자막 파일을 지정할 수 있습니다.

    참고:
    • 이것은베타수집 서비스-베타 참여에 관심이있는 경우 계정 관리자에게 문의하십시오.
    • XML 태그는 키워드를 사용하지만ooyala ,이 수집 방법을 사용하면 Video Cloud 계정에 비디오를 업로드 할 수 있습니다.

    XML 매니페스트 파일 만들기

    1. 비디오 및 기타 콘텐츠 (썸네일, 폐쇄 캡션 파일 등)를 호스팅 서버에 업로드합니다. URL은 공개적으로 사용할 수 있어야합니다.
    2. XML 유효성을 검사하는 텍스트 편집기를 엽니 다. (Brightcove 학습 서비스 팀은 Visual Studio 코드그러나 사용할 수있는 다른 몇 가지 좋은 옵션이 있습니다.)
