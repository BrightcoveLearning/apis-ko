개요

노트: 클라이언트 자격 증명을 얻는 기본 방법은 Studio Admin UI를 사용하는 것입니다. 이 문서는 curl을 사용하기로 선택한 사람들을 위한 것입니다.

이 문서의 정보를 이해하는 가장 좋은 첫 번째 단계는 OAuth 개요아직 하지 않은 경우.

메모: 문서의 단계는 다음과 같습니다. curl및 Mac 및 Linux 시스템에 고유하지만 Windows 운영 체제의 일부가 아닌 기타 명령줄 도구. Windows를 사용하는 경우 다음을 설치하여 이러한 도구를 얻을 수 있습니다. 시그윈 . 설치하는 동안 설치할 패키지를 묻는 메시지가 표시됩니다. 열기그물패키지를 확인한 다음곱슬 곱슬하다옵션을 선택하고 설치를 마칩니다. 이 작업을 완료하면 단계별로 수행할 수 있습니다. 플레이어 관리 문서가 cygwin 명령줄을 사용하여 작성된 대로 성공적으로 작성되었습니다. 도구를 사용하여 일반 Windows 명령줄에서 curl을 실행할 수 있지만 이 문서에서는 자세히 설명하지 않습니다.
