소개

노트:
  • 정책 키는 Video Cloud에 저장된 비디오를 제어하는 데만 사용됩니다. 독립 실행형 Brightcove 플레이어를 사용하는 경우 정책 키에 액세스하거나 검색할 필요가 없습니다.
  • 새 Brightcove 플레이어에 대해 생성된 정책 키는 기본적으로 검색, 도메인 제한, 지리적 필터링 또는 SSAI 요구에 대해 활성화되지 않습니다. Policy API(또는 이 샘플 앱)를 사용하여 수동으로 설정해야 합니다. 자세한 내용은정책 API 개요문서.
이 샘플 및 관련 코드는 프로덕션 개발을 위한 가이드로 제공됩니다. 코드는 프로덕션 용도로 인증되지 않았으며 공식적으로 지원되지 않습니다.
