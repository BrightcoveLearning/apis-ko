소개

Ingest Profiles API를 사용하면 한 번에 하나의 계정에서만 사용자 지정 수집 프로필을 만들 수 있습니다. 그러나 계정 목록을 반복하고 각 계정에 프로필을 추가하는 것은 간단합니다.

노트:
  • 앱에서는 사용자 지정 프로필만 추가할 수 있습니다. 표준 프로필은 이미 모든 계정에 있어야 합니다.
  • 계정에 추가하려는 프로필과 이름이 같은 프로필이 이미 있는 경우 새 프로필은 ~ 아니다추가되거나 기본 프로필로 설정됩니다.
이 샘플 및 관련 코드는 프로덕션 개발을 위한 가이드로 제공됩니다. 코드는 프로덕션 용도로 인증되지 않았으며 공식적으로 지원되지 않습니다.

자격 증명 가져오기

Ingest Profiles API를 사용하려면 적절한 자격 증명이 필요합니다.

대부분의 경우 자격 증명을 얻는 가장 쉬운 방법은 Studio Admin API 인증 섹션을 사용하는 것입니다(계정에 대한 관리자 권한 필요). 보다 API 인증 자격 증명 관리자세한 내용은. 대부분의 경우 모든 Ingest Profiles API 작업에 대한 권한을 얻고 싶을 것입니다.

필요한 권한을 Studio에서 사용할 수 없거나 OAuth API에서 직접 권한을 가져오려면 원하는 권한을 사용하십시오. 클라이언트 자격 증명 가져오기아래 나열된 문서. 어떤 옵션을 선택하든 올바른 작업 권한을 요청해야 합니다. cURL 또는 Postman과 함께 다음을 사용하여 Ingest Profiles API에 대한 모든 권한을 얻을 수 있습니다.

      "operations": [
        "video-cloud/ingest-profiles/profile/read",
        "video-cloud/ingest-profiles/profile/write",
        "video-cloud/ingest-profiles/account/read",
        "video-cloud/ingest-profiles/account/write"
      ]
  • OAuth API 샘플: 클라이언트 자격 증명 생성
  • 인증: cURL을 사용하여 클라이언트 자격 증명 가져오기
  • 인증: Postman을 사용하여 클라이언트 자격 증명 가져오기
