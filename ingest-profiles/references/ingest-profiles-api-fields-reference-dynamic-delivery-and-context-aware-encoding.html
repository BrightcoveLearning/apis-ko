동적 프로필 옵션 필드

동적 프로필 옵션 필드는 컨텍스트 인식 인코딩을 사용할 때 변환 생성에 대한 지침을 제공합니다.

동적 프로필 옵션
재산 유형 설명 DD CAE 기본
video_codecs 정렬 혼합 코덱 출력의 경우 변환을 생성할 코덱의 배열입니다. 현재 허용되는 항목 값은h264그리고hevc [2-1] 없음

  • min_renditions필수의
  • min_bitrate
  • max_bitrate
  • max_first_rendition_bitrate
  • min_resolution
  • max_resolution

참고: 이 설정은 전역 수준에서도 사용됩니다. 전역 및 코덱별 수준에서 설정을 사용하는 경우 값이 호환되어야 합니다. 예를 들어 전역 수준에서 설정된 max_renditions는 코덱별 수준에서 설정된 min_rendition 값의 합보다 크거나 같아야 합니다.

    • 워터마크 필드

    워터마크 이미지는 오른쪽 하단 모서리의 각 비디오 프레임에 레코딩됩니다.

    워터마크는 아래 표시된 필드를 통해 CAE 프로필에 지정할 수 있습니다. 정적 Dynamic Delivery 수집 프로필에 워터마크를 추가하려면 Brightcove 지원에 문의하십시오.
    재산 유형 설명 기본
    url 끈 이미지의 URL 없음
    width 끈[3-2] 워터마크의 너비(픽셀 또는 프레임 너비의 백분율) - 예20%[3-1] 없음
    height 끈[3-2] 워터마크의 높이(픽셀 또는 프레임 높이의 백분율) - 예10%[3-1] 없음
    x 끈[3-2] 비디오 프레임의 왼쪽 가장자리에서 워터마크 중심까지의 거리(픽셀 또는 프레임 너비의 백분율) 없음
    y 끈[3-2] 비디오 프레임의 상단 가장자리에서 워터마크 중심까지의 거리(픽셀 또는 프레임 너비의 백분율) 없음

    노트

    • [3-1] 너비만 지정하면 이미지 종횡비가 자동으로 유지됩니다.
    • [3-2] 비록width , height , x , 그리고y필드는 문자열 값입니다. 숫자(픽셀 또는 백분율)가 포함되어 있습니다.
