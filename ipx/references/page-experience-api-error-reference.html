In-Page Experience API 오류

In-Page Experience API 오류 참조
오류 예 설명
400 BAD_REQUEST 
{
  "status": 400,
  "code": "BAD_REQUEST",
  "message": "No template specified"
}
{
  "status": 400,
  "code": "BAD_REQUEST",
  "message": "Specified template does not exist"
}
{
  "status": 400,
  "code": "BAD_REQUEST",
  "details": [
    "Path `name` is required."
  ]
}
 일반적으로 생략과 같은 요청 본문에 문제template또는 기타 필수 필드 또는 존재하지 않는 템플릿 지정
401 UNAUTHORIZED 
[
  {
    "error_code": "UNAUTHORIZED",
    "message": "Permission denied."
  }
]
 요청과 함께 액세스 토큰이 전송되지 않았거나 액세스 토큰을 만드는 데 사용된 클라이언트 자격 증명에 요청에 대한 올바른 권한이 없습니다.
404 NOT_FOUND 
{
  "status": 404,
  "code": "NOT_FOUND",
  "message": "Experience {experience_id} not found"
}
 업데이트하려는 경험이 존재하지 않습니다. 삭제되었거나 요청의 경험 ID가 올바르지 않을 수 있습니다.
