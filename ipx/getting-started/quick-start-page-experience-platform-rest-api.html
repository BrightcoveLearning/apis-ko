소개

In-Page Experience API를 사용하면 프로그래밍 방식으로 인페이지 경험을 관리할 수 있으므로 CMS 통합에 특히 유용하므로 사용자가 Studio의 갤러리로 이동하는 대신 자신의 CMS에서 관리 경험을 만들 수 있습니다.

이 빠른 시작은 사용자가 다음을 수행할 수 있는 샘플 구현을 제공합니다.

  1. 계정에 대한 모든 인페이지 경험 템플릿 가져오기
  2. 선택한 템플릿에서 기본 경험 만들기
  3. 경험에 추가할 재생 목록 또는 비디오 목록을 선택하십시오.
  4. 선택한 비디오/재생 목록으로 경험 업데이트
  5. 경험 게시

아래는 앱 로직의 상위 레벨 보기입니다.

코드펜 사용하기

다음은 위의 CodePen을 효과적으로 사용하기 위한 몇 가지 팁입니다.

  • 클릭하여 플레이어의 실제 표시를 전환합니다. 결과단추.
  • 클릭 HTML/CSS/JS버튼을 눌러 코드 유형 중 하나를 표시합니다.
  • 딸깍 하는 소리 CodePen에서 편집이 CodePen을 자신의 계정으로 포크하려면 오른쪽 상단 모서리에 있습니다.
  • 이 샘플과 관련된 모든 코드 찾기 GitHub 저장소 .
