Brightcove Syndication Feed API는 다음을 사용하여 생성된 MRSS 피드를 검색하기 위한 공개 API입니다. 신디케이션 구성 API .

Syndication Configuration API를 사용하여 간단히 피드를 검색할 수도 있습니다. syndication_url신디케이션 개체에서 브라우저로 반환되지만 Feed API는 인증이 필요 없다는 점에서 다릅니다. 올바른 URL을 가진 사람은 누구나 피드를 검색할 수 있습니다.

한 API 또는 다른 API를 사용하여 피드를 검색하는 기본 사용 사례는 다음과 같습니다.

  • 예를 들어 구독자의 경우 피드를 직접 호스팅하려는 경우 신디케이션 구성 API를 사용하여 피드를 검색합니다.
  • 피드를 제공하여 원하는 방식으로 호스팅하고 싶은 신디케이션 파트너가 있는 경우 다음을 사용하여 Feed API URL을 제공할 수 있습니다. syndication_id피드를 직접 검색합니다.

API 참조/기본 URL

NS 피드 API 참조사용 가능한 요청 및 응답에 대한 전체 세부 정보가 포함되어 있습니다.

기본 URL은 다음과 같습니다.

https://social.feeds.brightcove.com/v1

입증

Feed API에 대한 액세스는 ~ 아니다피드는 공개적으로 액세스할 수 있으므로 인증이 필요합니다.

운영

Feed API에 대한 작업은 하나만 있습니다. 다음을 기반으로 피드를 가져옵니다. syndication_id .

요청 URL

    https://social.feeds.brightcove.com/v1/accounts/{account_id}/mrss/syndications/{syndication_id}/feed

방법: 가져 오기

쿼리 매개변수

모든 쿼리 매개변수는 선택사항입니다.

limit
피드 응답에 포함할 최대 항목 수입니다. 허용되는 범위는 1에서 100입니다. 기본: 100 .
offset
반환된 피드를 생성할 때 건너뛸 항목 수입니다. 기본: 0 .
updated_after
ISO-8601 날짜/시간 문자열(예: 2016-03-16T15:42:06.000Z 또는2016-03-16T10:47:29-05:00 ) 지정된 날짜 이후에 업데이트된 자산으로 피드 결과를 제한합니다.

요청 본문: 없음

이것은 중요합니다! 요청 본문을 포함하면 오류가 반환됩니다.

응답 유형: XML
