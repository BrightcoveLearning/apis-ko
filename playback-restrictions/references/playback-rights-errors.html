오류 참조

다음 상태 목록 및 오류 코드는 Brightcove Playback Restrictions 작업 시 문제를 디버그하는 데 도움이 됩니다.

상태 에러 코드 오류 하위 코드 설명
400 BAD_REQUEST DUPLICATE_PARAMETERS 요청에 동일한 매개변수 이름이 두 번 이상 제공되었습니다.
401 ACCESS_DENIED TOKEN_REQUIRED NS JSON 웹 토큰 (JWT)는 재생 권한에 필요하지만 제공되지 않았습니다.
401 ACCESS_DENIED TOKEN_VALIDATION NS JSON 웹 토큰 (JWT) 제공된 유효성 검사를 통과하지 못했습니다.
403 FORBIDDEN ACCOUNT_ID 정책 키의 계정 ID가 API 요청의 계정과 일치하지 않습니다.
403 FORBIDDEN CLIENT_GEO 비디오는 현재 지역에서 재생이 제한됩니다. 메시지에는 특정 문제에 대한 추가 정보가 포함됩니다.
403 FORBIDDEN CLIENT_IP 동영상은 현재 IP 주소로 제한됩니다.
403 FORBIDDEN DOMAIN 비디오는 현재 도메인에서 재생이 제한됩니다.
403 FORBIDDEN OFFERS NS JSON 웹 토큰 (JWT)는 비디오에서 제공되는 제안을 기반으로 한 재생을 허용하지 않습니다.
403 FORBIDDEN TAGS NS JSON 웹 토큰 (JWT)는 비디오에서 사용 가능한 태그를 기반으로 한 재생을 허용하지 않습니다.
403 FORBIDDEN VIDEO_NOT_PLAYABLE 단일 비디오 요청에 대해 비디오가 존재하지만 지금은 재생할 수 없습니다.
403 FORBIDDEN Denied by stream limiting (create) 제한된 스트림이 최대 시청자 수에 도달했습니다.
403 FORBIDDEN Denied by stream limiting (renew) 제한된 스트림이 이미 최대 시청자 수에 의해 시청되고 있습니다.
403 FORBIDDEN Denied by stream limiting 제한된 스트림이 최대 시청자 수에 도달했습니다.
403 FORBIDDEN Denied by device limiting 제한된 스트림이 최대 고유 등록 장치 수에 도달했습니다.
404 NOT_FOUND VIDEO_NOT_FOUND 요청한 동영상을 사용할 수 없습니다
404 NOT_FOUND PLAYLIST_NOT_FOUND 요청한 재생 목록을 사용할 수 없습니다
405 METHOD_NOT_ALLOWED METHOD_NOT_ALLOWED 이 API에는 GET, HEAD 및 OPTIONS만 허용됩니다.
500 SERVER_ERROR SERVER_ERROR 인터넷 서버 오류
502 SERVER_ERROR ACCOUNT_RETRIEVE_FAILURE 백엔드 서버에서 잘못된 응답을 받았습니다.
502 SERVER_ERROR VIDEO_RETRIEVE_FAILURE 비디오를 사용할 수 없음
502 SERVER_ERROR VIDEO_URLS_RETRIEVE_FAILURE 스트림 URL을 사용할 수 없음
502 SERVER_ERROR PLAYLIST_RETRIEVE_FAILURE 동영상 재생목록을 사용할 수 없음
502 SERVER_ERROR PLAYBACK_RIGHT_RETRIEVE_FAILURE CDN에서 재생 권한을 검색할 수 없습니다.
502 SERVER_ERROR PLAYLIST_VIDEOS_RETRIEVE_FAILURE 재생목록 동영상을 사용할 수 없음
502 SERVER_ERROR LICENSE_RETRIEVE_FAILURE 이 비디오에 대한 라이센스 서버에서 라이센스를 가져오지 못했습니다.
502 SERVER_ERROR OFFERS_RETRIEVE_FAILURE 이 동영상에 대한 API에서 제안을 가져오지 못했습니다.
502 SERVER_ERROR RIGHTS_RETRIEVE_FAILURE 이 비디오에 대한 재생 권한을 검색하지 못했습니다.
503 SERVICE_UNAVAILABLE 백엔드 서버에서 이 응답을 반환했습니다.
504 SERVER_TIMEOUT 백엔드 서버 또는 의존하는 서버 중 하나가 시간 초과되었습니다.
