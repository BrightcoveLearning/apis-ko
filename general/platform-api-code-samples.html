라이브 이벤트 동시 시청자

이 샘플은 Get Live Analytics 시계열 엔드포인트를 사용합니다. Analytics API다른 시간 간격을 기반으로 한 라이브 이벤트의 동시 시청자에 대한 보고서를 생성합니다.

일일 분석 보고서 생성기

현재 날짜까지 데이터를 사용할 수 있는 가장 빠른 날짜에 대한 계정의 모든 비디오에 대한 일일 분석 보고서를 CSV 형식으로 생성합니다.

목적지 보고서

두 개의 분석 보고서를 생성합니다. 1) 비디오를 본 모든 URL의 목록 2) 비디오를 본 모든 플레이어의 목록과 플레이어가 있는 도메인.

지리적 분석 보고서

Analytics API를 사용하여 국가별 조회수 보고서 생성

국가 및 날짜별 분석

이 샘플에서는 선택한 기간 동안 매일 국가별 총 동영상 조회수를 보여주는 보고서를 만드는 방법을 배웁니다. 반환된 데이터를 날짜 및/또는 국가별로 필터링할 수도 있습니다.

동영상 참여 그래프

계정의 동영상에 대한 참여 그래프를 생성합니다.

플레이어 및 요일별 동영상 보고서

이 예에서는 계정의 모든 비디오에 대한 보고서를 플레이어별로 그룹화하여 생성합니다.

일별 순 사용자

이 예에서는 계정에 대한 일별 순 사용자 및 동영상 조회수에 대한 보고서를 생성합니다.

태그별 동영상 보고서

이 샘플에서는 비디오에 할당된 태그별로 보고서를 가져오고 결과를 JSON 또는 CSV 형식으로 반환하는 방법을 배웁니다.

참조 ID별 동영상 보고서

이 샘플을 사용하면 동영상 참조 ID로 동영상에 대한 보고서를 얻을 수 있습니다.

CMS API 테스터

시작하기 CMS API몇 가지 일반적인 요청 유형과 해당 유형이 반환하는 데이터를 살펴봅니다.

여러 비디오에 사용자 정의 필드 추가

이 샘플에서는 사용자 정의 필드에 대한 UI를 만들고 여러 비디오에 사용자 정의 필드를 추가하는 방법을 배웁니다.

재생 목록별 분석

이 샘플에서는 사용 방법을 배웁니다. CMS API그리고Analytics API함께 계정에서 재생 목록을 선택하고 재생 목록의 모든 동영상에 대한 분석 데이터를 가져옵니다.

성능이 낮은 콘텐츠 식별

이 샘플에서는 사용 방법을 배웁니다. CMS API그리고Analytics API지난 1~6개월 동안 조회수가 5회 미만인 동영상 목록을 작성합니다.

최근에 추가된 동영상

이 샘플에서는 최근 기간(입력에서 정의)에 Video Cloud에 추가된 비디오에 대한 분석 데이터를 가져오는 방법을 배웁니다.

데이터 수집 샘플

이 샘플에서는 Brightcove 웹 또는 SDK 플레이어에서 볼 수 없는 비디오에 대한 분석 데이터를 얻을 수 있도록 데이터 수집기에 정보를 보내는 방법을 배웁니다.

큐 포인트 추가

이 샘플에서는 CMS API를 사용하여 프로그래밍 방식으로 비디오에 큐 포인트를 추가하는 방법을 배웁니다.

미디어 공유를 위해 여러 제휴 계정 추가

이 샘플은 CMS API JavaScript 및 프록시(PHP)를 통해 미디어 공유를 위한 여러 제휴 계정을 추가합니다.

여러 비디오 공유

이 샘플은 CMS API JavaScript 및 프록시(PHP)를 통해 여러 동영상을 하나 이상의 제휴 계정과 공유합니다.

보고서에서 공유된 미디어

이 샘플은 계정에서 공유된 동영상에 대한 CSV 형식의 보고서를 제공합니다.

보고서에 공유된 미디어

이 샘플은 Video Cloud 계정에 공유된 모든 비디오에 대한 보고서를 생성합니다.

비디오 업데이트

이 샘플에서는 CMS Ingest API를 사용하여 여러 동영상의 메타데이터를 업데이트하는 방법을 배웁니다.

비디오 보고서 생성기

이 샘플은 CMS API CSV 형식으로 계정의 모든 동영상에 대한 보고서를 생성합니다. 100개의 배치로 비디오 데이터를 가져오므로 API 시간 초과 또는 기타 오류가 있는 경우 데이터의 일부가 계속 유지됩니다.

지역 속성 업데이트

이 샘플에서는 계정의 모든 동영상에 대한 지역 필터링 속성을 업데이트하는 방법을 배웁니다.

다운로드 링크

이 샘플에서는 사용 방법을 배웁니다. CMS API비디오의 최고의 MP4 변환을 위한 다운로드 링크 테이블을 생성합니다.

iTunes용 RSS 팟캐스트 피드

이 샘플에서는 사용 방법을 배웁니다. CMS API Apple iTunes용 RSS 팟캐스트 피드를 생성합니다.

비디오 사이트맵 생성기

이 샘플은 CMS API를 사용하여 동영상 사이트맵을 생성하는 방법을 보여줍니다.

재생 목록의 MRSS 피드

이 샘플에서는 사용 방법을 배웁니다. CMS API MRSS 사양을 기반으로 MRSS 피드를 생성합니다.

MRSS 생성기

이 샘플에서는 사용 방법을 배웁니다. CMS API MRSS 사양을 기반으로 MRSS 피드를 생성합니다.

비디오 검색 앱

이 샘플에서는 CMS API를 사용하여 비디오 검색 애플리케이션을 만드는 방법을 배웁니다. 액세스 토큰을 가져오고 API를 호출하기 위해 Brightcove Learning Services에서 구축한 프록시를 사용합니다.

비디오 스키마 빌더

이 샘플은 Microdata 또는 JSON-LD 형식으로 VideoObject 스키마를 생성합니다. 비디오에 스키마를 포함하면 검색 엔진 인덱서에 정보를 제공하고 SEO를 향상시킵니다.

비디오 변환 검사기

이 주제에서는 사용 방법을 배웁니다. CMS API계정의 모든 비디오에 대한 변환을 검사합니다.

전달 유형별 동영상 보고서

이 주제에서는 사용 방법을 배웁니다. CMS API다양한 전송 유형의 동영상에 대해 보고합니다.

오디오 전용 콘텐츠에 대한 보고서

비디오 라이브러리에 오디오 전용 콘텐츠가 있는 경우 해당 오디오 전용 비디오만 표시하는 보고서를 실행하는 것이 유용할 수 있습니다. 이 샘플은 그 방법을 보여줍니다.

예약된 비디오 추가

이 샘플에서는 비디오가 나중에 사용 가능하도록 예약하여 실수로 재생되기 전에 재생되지 않도록 하는 방법을 배웁니다.

Dynamic Ingest API 테스터

이 빠른 시작은 Dynamic Ingest API를 통해 사용할 수 있는 작업에 익숙해지는 데 도움이 됩니다.

텍스트 트랙 추가

이 샘플에서는 Dynamic Ingest를 사용하여 기존 비디오에 자산을 추가하는 앱을 빌드하는 방법을 배웁니다.

우선 순위가 낮은 비디오 수집

이 샘플에서는 여러 수집 작업을 낮은 우선 순위 대기열로 보내는 방법을 배웁니다.

비디오 수집(동적 전송)

이 샘플에서는 Dynamic Ingest API를 사용하여 여러 비디오를 수집하는 방법을 배웁니다. 액세스 토큰을 가져오고 API를 호출하기 위해 Brightcove Learning Services에서 구축한 프록시를 사용합니다.

비디오 재변환(Dynamic Delivery)

이 샘플에서는 Dynamic Ingest API를 사용하여 여러 비디오를 다시 코드 변환하는 방법을 배웁니다. 액세스 토큰을 가져오고 API를 호출하기 위해 Brightcove Learning Services에서 구축한 프록시를 사용합니다.

비디오 교체(동적 전달)

이 샘플에서는 Dynamic Ingest API를 사용하여 동영상을 교체하는 방법을 배웁니다. API는 여러 비디오에 대한 작업을 지원하지 않습니다. 여기에 있는 코드는 단순히 비디오를 하나씩 교체하기 위해 데이터 세트를 반복합니다.

사용자 지정 동적 배달 프로필 만들기

이 샘플에서는 Ingest Profiles API를 사용하여 사용자 지정 Dynamic Delivery 프로필을 만드는 방법을 배웁니다.

사용자 정의 컨텍스트 인식 인코딩 프로필 만들기

이 샘플에서는 Ingest Profiles API를 사용하여 사용자 지정 컨텍스트 인식 인코딩 프로필을 만드는 방법을 배웁니다.

계정에 대한 모든 프로필 가져오기

이 샘플은 계정에 대한 모든 수집 프로필을 검색하고 표시하고 싶지 않은 일부 항목을 필터링하는 방법을 보여줍니다.

기본 프로필 설정

이 코드 샘플에서는 Ingest Profiles API를 사용하여 여러 계정에 대한 기본 수집 프로필을 설정하는 방법을 배웁니다.

여러 계정에 프로필 추가

이 샘플 앱을 사용하면 여러 Video Cloud 계정에 맞춤형 수집 프로필을 추가할 수 있습니다. 새 프로필을 만들어 모든 계정에 추가하거나 한 계정에서 프로필을 선택하여 다른 계정에 추가할 수 있습니다.

클라이언트 자격 증명 생성

이 샘플을 사용하면 하나 이상의 계정에 대한 클라이언트 자격 증명을 생성하여 선택한 API 권한을 할당할 수 있습니다.

액세스 토큰 받기

이 샘플은 웹 앱의 Brightcove OAuth API에서 액세스 토큰을 검색하여 프록시를 통해 요청을 보냅니다.

재생 목록의 JSON/MRSS 피드

이 샘플에서는 Playback API를 사용하여 재생 목록을 검색하고 데이터에서 JSON 또는 MRSS 비디오 피드를 생성하는 방법을 배웁니다.

MRSS 생성기

이 샘플에서는 Playback API를 사용하여 MRSS 사양을 기반으로 MRSS 피드를 생성하는 방법을 배웁니다.

정책 키 생성

이 빠른 시작에서는 간단한 웹 앱을 사용하여 정책 키를 만듭니다. 키를 검색할 수 있는 옵션이 있습니다.

Ingest Profiles API 시작하기

자신의 계정 또는 샘플 Brightcove 계정에서 다양한 API 요청을 시도하십시오.

플레이어 만들기

이 예에서는 계정 정보를 입력합니다. 또한 미디어 및 미디어 유형 또는 Video Cloud 비디오 ID에 대한 URL을 입력합니다. 버튼을 클릭하면 플레이어가 생성되고 플레이어에 대한 선택 정보가 표시됩니다.

디스플레이 플레이어

샘플은 Brightcove 계정의 모든 플레이어 목록을 검색하고 플레이어의 일부 세부 정보와 페이지에 미리 보기를 표시하는 방법을 보여줍니다.

마지막으로 본 플레이어 통계

플레이어를 사용한 보고last_viewed data. 각 플레이어는 플레이어가 로드된 횟수에 대한 레코드가 포함된 배열과 연결되어 있습니다. 이 정보는 배열 이름에 저장됩니다. last_viewed .

플레이어 삭제

샘플은 모든 플레이어 목록을 검색하고 사용자가 선택한 플레이어를 삭제하는 방법을 보여줍니다(실제로 플레이어를 삭제하지는 않지만 방법을 보여줍니다).

GET 요청에 대한 Python 코드

샘플은 CMS API를 사용하여 비디오를 검색하는 방법을 보여줍니다.

동적 수집을 위한 Python 코드

샘플은 Video Cloud 비디오를 만들고 비디오 소스 파일을 수집하는 방법을 보여줍니다.

빠른 시작: Social API

이 빠른 시작을 통해 자신의 계정이나 Brightcove Learning Services 샘플 계정을 사용하여 Social API에 대해 사용 가능한 요청과 응답을 볼 수 있습니다.
