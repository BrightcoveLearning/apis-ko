자격 증명 얻기

얻으려면client_id그리고client_secret , OAuth UI로 이동하여 이 앱을 등록해야 합니다.

  • 브라이트코브 플레이어: //studio.brightcove.com/products/player/admin/oauthsettings
  • 비디오 클라우드: //studio.brightcove.com/products/videocloud/admin/oauthsettings

CURL 또는 Postman을 통해 자격 증명을 얻을 수도 있습니다. 다음을 참조하세요.

  • CURL을 사용하여 클라이언트 자격 증명 가져오기
  • Postman을 사용하여 클라이언트 자격 증명 가져오기

코드펜 사용하기

다음은 위의 CodePen을 효과적으로 사용하기 위한 몇 가지 팁입니다.

  • 클릭하여 플레이어의 실제 표시를 전환합니다. 결과단추.
  • 클릭 HTML/CSS/JS버튼을 눌러 코드 유형 중 하나를 표시합니다.
  • 딸깍 하는 소리 CodePen에서 편집이 CodePen을 자신의 계정으로 포크하려면 오른쪽 상단 모서리에 있습니다.
  • 이 샘플과 관련된 모든 코드 찾기 GitHub 저장소 .

프록시 코드

이 페이지에서 고유한 버전의 샘플 앱을 빌드하려면 고유한 프록시를 만들고 호스팅해야 합니다. (Brightcove Learning Services에서 사용하는 프록시는 Brightcove 도메인의 요청만 수락합니다.) 두 가지 버전의 프록시 코드를 다운로드할 수 있습니다.

  • 이것은 클라이언트 자격 증명이 요청과 함께 전달될 것으로 예상하는 일반 버전입니다.
  • 이 버전을 사용하면 25-26행의 프록시 자체에 클라이언트 자격 증명을 저장할 수 있습니다(권장).
