페이지 내용

    인제스트 프로파일 필드 참조

    이는 레거시 인제스트 시스템의 인제스트 프로파일 필드에 대한 참조입니다.
    이 가이드는레거시 인제스트 시스템을위한 것입니다. 대부분의 고객은동적 전달 시스템을사용하고 있습니다.

    프로파일 필드

    사용자가 검색된 콘텐츠를 사용하여 해당 프로필을 업데이트 할 의도로 프로그래밍 방식으로 프로필을 가져 오는 경우 사용자는 프로필을 삭제하고 " id "변환 용이지만 프로필에는 해당되지 않음" version "을 완전히 제거해야합니다.
    부동산 유형 설명 기본값
    id 끈 프로필 ID (읽기 전용) 없음
