개요

이 자습서에서는 API 수준에서 라이브 스트리밍 작업을 만드는 방법을 설명합니다. 브라이트코브 라이브그런 다음 생성 RTMP출력하여 YouTube Live 및 Facebook Live로 다시 스트리밍합니다.

노트:
  • 라이브 모듈에서 RTMP 출력을 사용하는 방법에 대한 정보는 다음을 참조하십시오. 라이브 RTMP 출력을 사용하여 Facebook 및 YouTube로 스트리밍 .
  • RTMP 출력하다지원하다rtmps URL - 사용해야 합니다. rtmps소셜 플랫폼이 지원하는 경우.
  • Facebook은 지원을 종료했습니다. rtmp .streams 2019년 5월 1일 현재 Facebook의 경우해야하다사용하다RTMPS .
  • NS RTMP URL에 대해 현재 지원되는 형식만 ~이다rtmps://host:port/application/streamName .

이 튜토리얼에서는곱슬 곱슬하다 API 요청을 만들 수 있지만불명 증 , 우편 집배원또는 대신 다른 REST 클라이언트.

인코더 구성

여기에 표시된 단계는 다음을 가정합니다. 엘리멘탈 라이브 박스이 자습서의 단계를 테스트하는 데 사용된 인코더입니다. 다른 인코더가 있는 경우 설정이 비슷해야 합니다.

이 경우 가장 중요한 설정은 다음과 같습니다.

  • 타임코드 소스를 "시스템 시계"로 구성
  • "OnFi 타임코드 주파수"를 1로 설정
  • 영상 내 "타임코드 삽입" 확인
  • 권장: 프레임 속도 = 소스 팔로우

YouTube 라이브에 라이브 스트림 추가

여기에서는 라이브 스트리밍이 활성화된 YouTube 계정이 있다고 가정합니다.

  1. 딸깍 하는 소리생중계 , 다음을 확인해야 합니다.

  2. "인코더 라이브 스트리밍"을 클릭하십시오.

  3. 라이브 이벤트 YouTube 메타데이터를 구성하고 "인코더 설정" 데이터를 사용하여 Brightcove 라이브 스트림을 YouTube에 연결합니다.

  4. 이제 YouTube 라이브가 설정되었으므로 라이브 작업에서 RTMP 출력을 생성합니다.

  5. 복사곱슬 곱슬하다아래 명령을 텍스트 편집기로 이동합니다.

    curl -X POST \
  https://api.bcovlive.io/v1/jobs/{{bcov-live-job-id}}/rtmpouts \
  -H 'Content-Type: application/json' \
  -H 'x-api-key: {{bcov-live-api-key}}' \
  -d '{
  "url": "rtmp://a.rtmp.youtube.com/live2/{{youtube-secret-stream-name}}"
}'

  6. 바꾸다:

    • {{bcov-live-job-id}} Brightcove 라이브 작업 ID로. 이 예에서 이 값은3b6871bf2f344acaa6b397d09b476018 - 라이브 작업 생성 요청에 대한 응답에서 귀하의 가치가 달라집니다.
    • {{bcov-live-api-key}} Brightcove 라이브 API 키용. 예를 들어: abcdfeg-this-is-a-fake-api-key-FgJajjasd12hJHsZ
    • {{youtube-secret-stream-name}} YouTube에서 제공하는 스트림 이름/키
  7. 편집한 코드를 명령줄 앱에 붙여넣고 실행합니다.

  8. 응답은 다음과 같아야 합니다.

    {
          "connection_info": {
            "host": "a.rtmp.youtube.com",
            "port": 1935,
            "application": "live2",
            "streamName": "{{youtube-secret-stream-name}}"
          },
          "duration_history": [],
          "rtmp_out_id": "27aacdfa5bcf4436ade0519e6a748aba",
          "stream_start": 1542902218238
        }

  9. YouTube 재생을 테스트하려면 YouTube로 이동하세요. 내 채널 :

Facebook 라이브에 라이브 스트림 추가

여기에서 라이브 스트리밍이 활성화된 Facebook 계정이 있다고 가정합니다.

  1. 딸깍 하는 소리라이브 비디오

  2. 딸깍 하는 소리연결하다 Brightcove 라이브 작업을 연결하는 데 필요한 데이터를 가져오는 스트림 키:

  3. 다음으로 Facebook의 "서버 URL" 및 "스트림 키"를 사용하여 Brightcove 라이브 작업에 대한 라이브 출력을 만듭니다. 복사곱슬 곱슬하다아래 명령을 입력하고 텍스트 편집기에 붙여넣습니다.

    curl -X POST \
  https://api.bcovlive.io/v1/jobs/{{bcov-live-job-id}}/rtmpouts \
  -H 'Content-Type: application/json' \
  -H 'x-api-key: {{bcov-live-api-key}}' \
  -d '{
  "url": "rtmp://live-api-s.facebook.com:80/rtmp/{{facebook-secret-stream-name}}"
}'

  4. 바꾸다:

    • {{bcov-live-job-id}} Brightcove 라이브 작업 ID로. 이 예에서 이 값은3b6871bf2f344acaa6b397d09b476018 - 라이브 작업 생성 요청에 대한 응답에서 귀하의 가치가 달라집니다.
    • {{bcov-live-api-key}} Brightcove 라이브 API 키로. 예를 들어: abcdfeg-this-is-a-fake-api-key-FgJajjasd12hJHsZ
    • {{facebook-secret-stream-name}} Facebook에서 제공하는 스트림 이름과 함께

  5. 응답은 다음과 같아야 합니다.

    {
  "connection_info": {
      "host": "live-api-s.facebook.com",
      "port": 80,
      "application": "rtmp",
      "streamName": "{{facebook-secret-stream-name}}"
  },
  "duration_history": [],
  "rtmp_out_id": "59323c3d1e2b44748755853193fb126c",
  "stream_start": 1542904101097
}

  6. Facebook 라이브 스트림을 테스트하려면 Facebook 웹페이지에서 "라이브 시작"을 클릭하십시오.
