샘플에 대한 참고 사항

이 샘플의 코드는 모두 JavaScript이며 두 부분이 있습니다(둘 모두 CodePen에서 찾을 수 있습니다. 플러그인 스크립트는 맨 위에 있고 플러그인을 플레이어에 추가하는 짧은 코드는 맨 아래에 있음).

  • videojs 플레이어에 플러그인을 추가하는 스크립트
  • Data Collector에 데이터를 보내는 플러그인용 스크립트

데이터가 Data Collection API로 "전송"되는 방식은 요청 URL을 다음으로 설정하는 것입니다. src새 이미지의 값( img ) 요소를 선택한 다음 해당 요소를 페이지에 삽입합니다.

보다는 같은 방법을 사용해야 합니다. XMLHttpRequest또는 피하기 위해 다른 CORS문제.
