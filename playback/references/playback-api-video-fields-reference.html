1 번 테이블: 기본 필드
들 유형 기본 설명
id 끈 없음 동영상 아이디
account_id 끈 없음 계정의 아이디
created_at ISO 8601 날짜-시간 문자열 없음 비디오가 계정에 추가된 날짜-시간 예시: "2014-12-09T06:07:11.877Z"
cue_points 의 배열 cue_point 개체 빈 배열 플레이어 API를 통해 생성될 미드롤 광고 요청 또는 기타 작업에 대한 마커
custom_fields 물체 빈 개체 사용자 정의 필드 이름:값 쌍의 맵; 값이 있는 필드만 포함됩니다.
description 끈 없는 비디오에 대한 간단한 설명 - 최대 250자의 단일 바이트
duration 숫자 없는 비디오 길이(밀리초)
economics 끈 AD_SUPPORTED 동영상에 대한 광고 요청이 허용되는지 여부를 나타냅니다. 값: AD_SUPPORTED또는FREE
labels 문자열 배열 없는 의 배열라벨비디오에 할당
link 링크 개체 없는 관련 링크
long_description 끈 없는 1바이트 최대 5000자
name 끈 없음 동영상 제목 - 필수 입력란(최대 1바이트 256자)
poster 끈 없는 포스터 이미지 URL
poster_sources 정렬 [] 비디오의 포스터에 해당하는 개체 배열입니다. 현재 하나의 포스터만 있으므로 하나의 개체가 있습니다. 객체에는 단일 속성이 있습니다. src값이 포스터 이미지의 URL입니다.
published_at ISO 8601 날짜-시간 문자열 없음 동영상이 게시된 날짜-시간 예시: "2014-12-09T06:07:11.877Z"
reference_id 끈 없는 계정 내에서 고유한 모든 값(1바이트 최대 150자)
sources 정렬 빈 배열 의 배열비디오 소스 개체
tags 정렬 빈 배열 비디오에 추가된 태그 배열(문자열)(태그당 최대 128자)
text_tracks 의 배열 text_track 개체 빈 배열 비디오와 연결된 WebVTT 캡션 데이터
thumbnail 끈 없는 썸네일 이미지 URL
thumbnail_sources 정렬 [] 비디오의 썸네일에 해당하는 객체의 배열입니다. 현재 썸네일이 하나뿐이므로 하나의 개체가 있습니다. 객체에는 단일 속성이 있습니다. src그 값은 썸네일 이미지의 URL입니다.
updated_at ISO 8601 날짜-시간 문자열 없음 동영상이 마지막으로 수정된 날짜-시간입니다.[1-2] 예시: "2015-01-13T17:45:21.977Z"
variants 정렬 [] 비디오에 대한 다국어 메타데이터를 포함하는 객체 배열

노트

  • [1-2] 사용자 또는 Video Cloud 시스템이 수정했을 수 있습니다.

key_systems물체

key_systems 개체에는 각 DRM 키 이름에 대해 하나씩 개체 목록이 포함되어 있습니다. 이 개체가 정의된 경우 해당 소스는 콘텐츠로 보호됩니다.

표 4: key_systems 개체
들 유형 기본 설명
키 이름 물체 없는 DRM 유형을 지정하는 개체의 이름 - 유효한 키에는 다음이 포함됩니다.

* 페어플레이: com.apple.fps.1_0
* 플레이 준비: com.microsoft.playready
* 와이드바인: com.widevine.alpha
FairPlay 사용key_request_url그리고certificate_url
다른 사람들이 사용license_url		 끈 없는 DRM 라이선스 또는 인증서가 있는 URL
 

예 1:

    "key_systems": {
      "com.apple.fps.1_0": {
        "key_request_url": "https://....",
        "certificate_url": "https://..."
      }
    }

예 2:

    "key_systems": {
      "com.microsoft.playready": {
        "license_url": "https://..."
      },
      "com.widevine.alpha": {
        "license_url": "https://..."
      }
    },

variant물체

각variant특정 언어에 대한 비디오 메타데이터 버전이 포함되어 있습니다.

표 5: 변형 개체
들 유형 기본 설명
language 끈 없음 language-COUNTRY 형식의 변형 언어(예: 자자
name 끈 없음 번역된 제목
description 끈 없음 번역된 설명
long_description 끈 없음 번역된 long_description
custom_fields 물체 {} 사용자 정의 필드에 대한 키-값 쌍의 맵

예시

{
  "language": "ja-JA",
  "name": "バーゼル：ライン川",
  "description": "スイス、バーゼルの中心部にあるライン川のパノラマビュー。",
  "long_description": "スイス、バーゼルの中心部にあるライン川のパノラマビュー。",
  "custom_fields": {
    "language": "Japanese"
  }
}
