기본 필드

1 번 테이블: 기본 필드
들 유형 기본 설명
id 끈 없음 재생 목록 ID
account_id 끈 없음 계정의 아이디
created_at ISO 8601 날짜-시간 문자열 없음 날짜-시간 비디오가 계정에 추가되었습니다. 예시: "2014-12-09T06:07:11.877Z"
description 끈 null 비디오에 대한 간단한 설명 - 최대 128자
favorite 부울 false 재생 목록이 즐겨 찾기에 포함되어 있는지 여부
limit 숫자 100 재생 목록이 보유할 수 있는 동영상 수(정수, 1 - 100) 스마트 재생 목록에 대해서만 반환됨
name 끈 없음 재생목록 제목 - 필수 입력란
reference_id 끈 null 계정 내에서 고유한 모든 값
search 끈 null 스마트 재생 목록의 동영상을 정의하는 데 사용되는 검색 문자열(예: +tags:"bird","birds" )
type 끈 재생 목록의 유형 참고 사항 참조[1] 아래에
updated_at ISO 8601 날짜-시간 문자열 없음 날짜-시간 재생 목록이 마지막으로 수정되었습니다. 예시: "2015-01-13T17:45:21.977Z"
videos 정렬 의 배열비디오 개체재생목록에 있는 동영상의 경우 재생 목록에 대해 처음 20개의 동영상만 반환됩니다.

노트

  • [1]

    NS type값은 다음 중 하나일 수 있습니다. 명백한, ACTIVATED_OLDEST_TO_NEWEST , ACTIVATED_NEWEST_TO_OLDEST , ALPHABETICAL , PLAYS_TOTAL , PLAYS_TRAILING_WEEK , START_DATE_OLDEST_TO_NEWEST , START_DATE_NEWEST_TO_OLDEST

    EXPLICIT수동 재생 목록임을 의미합니다. 다른 모든 유형은 순서 체계에 해당하는 이름을 가진 스마트 재생 목록입니다.
