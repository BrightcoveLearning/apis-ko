오류 메시지

Dynamic Ingest API에서 다음 오류 메시지를 반환할 수 있습니다.

동적 수집 오류 메시지
암호 메세지 유형 설명
400 PROFILE json

이름으로 프로필을 찾을 수 없습니다
400 BAD_REQUEST json

요청 본문을 구문 분석할 수 없습니다.
400 NOT_SUBMITTED json

작업을 제출할 수 없습니다. 나중에 다시 시도해 주세요.
400 NO_SUCH_VIDEO json

참조된 비디오를 찾을 수 없습니다
400 NO_SOURCE json

사용할 소스를 찾을 수 없습니다
400 CDN_CREDENTIALS json

CDN 자격 증명을 가져올 수 없습니다.
400 BAD_CALLBACKS json

콜백이 예상한 형식이 아닙니다.
401 UNAUTHORIZED json

인증 실패; 액세스 토큰에 대한 클라이언트 자격 증명이 올바른지 확인하십시오.
403 DYNAMIC_DELIVERY_NOT_ALLOWED json

이 계정은 Dynamic Delivery를 사용할 수 없지만 Dynamic Delivery 프로필이 지정되었습니다.
403 CONTEXT_AWARE_ENCODING_NOT_ALLOWED json

이 계정은 컨텍스트 인식 인코딩에 대해 활성화되어 있지 않지만 컨텍스트 인식 인코딩 프로필이 지정되었습니다.
404 RESOURCE_NOT_FOUND json

API가 요청한 리소스를 찾을 수 없습니다.
415 UNSUPPORTED_MEDIA_TYPE json

수집하려는 파일이 지원되는 유형이 아닙니다(참조지원되는 비디오 및 오디오 형식, 코덱 및 컨테이너 ) 또는 요청에Content-Type: application/json헤더
422 MALFORMED_SOURCE_URL json

소스 URL 형식이 잘못되었습니다.
422 BAD_PROTOCOL_SOURCE_URL json

소스 URL이 지원되지 않는 프로토콜을 사용합니다.
422 EXCEED_MAXIMUM_VTT_SOURCES json

vtt 소스가 최대 크기를 초과합니다.
422 INVALID_VTT_KIND json

vtt 종류가 잘못되었습니다.
422 CONSTRAINT_VIOLATION json

이미지 소스가 제공된 경우 capture-image가 허용되지 않습니다.
422 UNPROCESSABLE_ENTITY json

요청 데이터에 처리할 수 없는 엔터티가 있습니다.
429 RATE_LIMIT_EXCEEDED json

동적 수집 작업이 생성되지 않았습니다. 다시 시도하기 전에 이 계정의 동시 작업 수를 줄이십시오. 이 오류가 기록되었습니다
400 CDN_CONFIGS json

CDN 자격 증명을 가져올 수 없습니다.
400 AMBIGUOUS_REQUEST json

마스터 URL과 use_archived_master가 모두 요청에 설정되었습니다.
500 INGEST_ERROR json

이 오류는 일반적으로 수집 서비스가 일시적으로 중단되었음을 나타냅니다.
